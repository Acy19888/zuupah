================================================================================
                          SETUP INSTRUCTIONS
================================================================================

1. INSTALL DEPENDENCIES
   npm install

2. CONFIGURE FIREBASE
   - Copy .env.example to .env
   - Add your Firebase configuration values
   - Enable Auth, Firestore, and Storage in Firebase console

3. SETUP FIRESTORE
   - Create collections: books, users, firmwareVersions
   - Set security rules for data access
   - Add sample book data

4. RUN ON iOS
   cd ios && pod install && cd ..
   npm run ios

5. RUN ON Android
   npm run android

6. PUSH TO GITHUB
   git push -u origin main
   (Note: May require GitHub credentials setup)
